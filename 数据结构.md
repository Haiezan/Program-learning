# 算法

### 正确性
四个层次
* a 不含语法错误；
* b 程序对于几组数据能够得出满足要求的结果；
* c 对于精心选择的、典型的、苛刻且带有刁难性的几组输入数据能够得出满足要求的结果；
* d 对于一切合法的输入数据都能得出满足要求的结果。
